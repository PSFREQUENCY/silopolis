## What We Built

**SILOPOLIS** is a fully autonomous multi-agent economy deployed on X Layer. ## x402 Micropayments

Agents acquire skill relics via **HTTP 402 micropayment protocol**:

```
1. Agent needs "Oracle Lens" relic
2. Issues GET /api/skill/oracle-lens
3. Server responds: HTTP 402 + WWW-Authenticate: x402 ...
4. SILO-SKILL-3 intercepts 402 challenge
5. Swaps available token вҶ’ required payment token via OnchainOS
6. Replays request with X-Payment: <signed-receipt> header
7. Skill relic stored on-chain in SkillMarket.sol
8. Agent reputation updated: collaboration +1, proficiency unlock
```

This creates a live **agent micropayment economy** вҖ” agents earn OKB from trades, spend it to acquire skills, use those skills to trade better.

## Self-Healing & Safety

- **3 consecutive losses** вҶ’ 8-hour pause (circuit breaker in `core/risk.py`)
- **Threat score вүҘ 76** вҶ’ Gemini decision blocked by GUARD agent before execution
- **Daily OKB budget** вҶ’ hard cap per tier, resets every 24h
- **50% profit capture** вҶ’ every winning trade: 50% stays in vault, 50% compounds
- **Immutable audit trail** вҶ’ SQLite `decision_log` table, every decision timestamped
